The Federal Communications Commission (FCC) established Automated Frequency Coordination (AFC), a system to coordinate the 6 GHz band use in unlicensed Wi-Fi 7 (aka 802.11be) devices to play nice with existing licensed devices using the same 6 GHz band, such as incumbent fixed service stations. This website is used by Broadcom OEMs to manage Wi-Fi 7 access points containing Broadcom chips.

Login on this site is done via single sign-on (SSO) at the Broadcom Support website.

Process for Broadcom OEMs to request access to this Broadcom AFC website:

  • Upon approval you will get an email with instructions on how to get onboard
